# Проект "Игры Разума"

## Игра "Проверка на чётность"

Суть игры в следующем: пользователю показывается случайное число.
И ему нужно ответить yes, если число чётное, или no — если нечётное

При верном ответе отобразится: 'Correct!' \
При неверном ответе будет подсказка, что ответ неверен и просьба попытаться снова

### Запуск
`make brain-even`

## Игра "Калькулятор"

Суть игры в следующем: пользователю показывается случайное математическое выражение, 
например 35 + 16, которое нужно вычислить и записать правильный ответ.

При верном ответе отобразится: 'Correct!' \
При неверном ответе будет подсказка, что ответ неверен и просьба попытаться снова

    Достаточно реализовать следующие операции: +, - и *.
    Операции, как и числа, выбираются случайным образом.


### Запуск
`make brain-calc`

## Игра "Арифметическая прогрессия"

Суть игры в следующем: Показываем игроку ряд чисел, образующий арифметическую прогрессию, 
заменив любое из чисел двумя точками. Игрок должен определить это число.


    Рекомендуемая длина прогрессии – 10 чисел. Длина может генерироваться случайным образом, но должна содержать не менее 5 чисел
    Позиция спрятанного элемента каждый раз изменяется (выбирается случайным образом)


При верном ответе отобразится: 'Correct!' \
При неверном ответе будет подсказка, что ответ неверен и просьба попытаться снова

### Запуск
`make brain-progression`
